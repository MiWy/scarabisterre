# Scarabisterre: a Hugo Theme

Scarabisterre is a (quite rough at the moment) remix of two Hugo themes: the [Finisterre](https://github.com/jonathan-g/hugo-finisterre) and the [Scarab](https://github.com/jweisber/scarab).

To see an example in action [go here](https://miwy.github.io/).
